안녕하세요!
퍼블리셔 강혜경 입니다.

완성된 웹사이트를 보며 “이 화면을 내가 만들었다”는 성취감을 느끼는 순간, 퍼블리싱이 제 길이라는 확신이 들었습니다.

처음 코딩을 배울 땐 낯설고 어려웠지만, HTML과 CSS 한 줄 한 줄이 쌓이며 화면이 완성되는 과정에서 점점 ‘즐거움’이 ‘일상’이 되었습니다.

저는 단순히 예쁜 화면을 만드는 것을 넘어, 사용자가 편하게 이용할 수 있는 구조와 접근성 그리고 협업이 원활한 코드 구조를 고민합니다. 디자인 감각과 개발 이해도를 함께 갖춘 퍼블리셔로 성장하고 싶습니다.

현재는 퍼블리싱 업무를 중심으로 역량을 키워가고 있지만, 필요하다면 언제든지 새로운 기술을 배우고 확장할 준비가 되어 있습니다.

제 포트폴리오를 봐주셔서 감사합니다.

  • 프로젝트명: 한화이글스(메인/로그인)

    프로젝트 개요

    프로야구 구단 ‘한화이글스’ 브랜드를 기반으로 실제 서비스 구조를 참고해 메인 페이지와 로그인 페이지를 설계·퍼블리싱한 프로젝트입니다.

    역할 및 작업 범위

    - 2Depth 네비게이션 구조 설계 및 마우스오버 인터랙션 구현
    - 비디오 기반 히어로 섹션 구성
    - flex·grid를 조합한 컴포넌트 기반 퍼블리싱 구조 설계
    - JS 기반 탭 필터링 기능(data-filter) 구현 (스파이더샵 / 형지샵)
    - 무한루프 슬라이드(선수 이미지) 순수 CSS 애니메이션으로 구현
    - 로그인 페이지 UI 단일 파일 구조 분리(css/login.css), 입력필드 UX 개선
    - 반응형 레이아웃(PC·Mobile) 제작

    기술 스택

    HTML5, CSS3(Flex/Grid/Animation), JavaScript, Figma, VSCode

    제작 기간

    7일

    참여도

    100%

    - 제품 박스 hover 시 상세 설명 오버레이 인터랙션 구현

    - store-search(매장 찾기) 폼 UI 제작 및 검색 UX 구조 구성
