[ 데이터 관리 (변수 - 상자, 저장공간) 사용처 ]

숫자(Number) → 계산 (합, 평균, 퍼센트, 총액)

문자열(String) → 글자 데이터 저장/출력 (이름, 안내문, 설명)

불리언(Boolean) → 조건 분기 (로그인 여부, 합격 여부) // 불리언(Boolean)은 보통 조건문이랑 같이 씀

배열(Array) → 같은 종류 데이터 여러 개 저장 (점수들, 상품들)

객체(Object) → 하나의 대상 정보 묶음 (회원, 상품, 게시글)



[ 변수 선언 키워드 (상자종류 2가지) ]

let(바뀔 수 있는 값 저장)

let count = 1;
count = 2; // ✅ 가능
-------------------------------
const(절대 바뀌면 안 되는 값 저장 (상수)

const pi = 3.14;
pi = 3.14159; // ❌ 오류



[ 변수값 정리 ]

let text = "안녕";         // 문자열(String) - 글자, 문장을 저장할 때
let num = 123;             // 숫자(Number) - 계산이 필요한 값
let isLogin = true;        // 불리언(Boolean, true/false) - 참/거짓만 필요한 상황 && 비교 연산, 논리 연산 = 무조건 true / false
let fruits = ["사과","배"]; // 배열(Array) - 같은 종류의 데이터를 여러 개 저장할 때
let user = {name:"홍길동", age:20}; // 객체(Object) - 하나의 대상(사람, 상품 등)에 여러 속성을 묶어서 저장할 때
let empty = null;          // 값이 없음 - "값 없음"을 표현
let nothing;               // undefined - (아직 값이 안 들어간 상태)

[ 변수 관련 용어 정리 ]

✅ prompt()
👉 사용자에게 입력창을 띄워서 값을 받아오는 함수예요.

let value = prompt("당신의 이름을 입력하세요", "홍길동");

- 첫 번째 인자 "당신의 이름을 입력하세요" → 입력창에 보일 안내 문구
- 두 번째 인자 "홍길동" → 입력창에 기본으로 들어갈 값(안 써도 됨)

✅ Number()

👉 괄호 안에 들어간 값을 숫자로 변환하는 함수예요.

왜 필요하냐?
→ prompt()가 돌려주는 값은 항상 "문자"이기 때문이에요.


👉 그래서 Number(prompt(...))는
“입력창에서 값 받아와서, 그걸 숫자로 바꿔 변수(상자)에 저장” 이라는 뜻이에요.

✅ 차이점 쉽게 비유

if문 = “조건에 따라 행동을 다르게 하기”

true/false 출력 = “조건의 판단 결과 자체를 보여주기”

즉,

if문은 “합격/불합격” 같은 문구를 직접 보여주고 싶을 때

true/false 출력은 “내가 짠 조건이 맞는지 확인하거나, 결과를 디버깅(테스트)”할 때 많이 씁니다.


✅ 정리

if → 사용자에게 친절한 결과 문구 보여주기 (ex: "합격입니다")

true/false 출력 → 프로그램 입장에서는 값이 제대로 계산됐는지 확인하는 테스트용
