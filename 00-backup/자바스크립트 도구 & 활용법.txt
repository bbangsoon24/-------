✅ 자바스크립트 도구 & 활용법
1. 변수 (데이터 저장 도구)

👉 데이터를 담아두고 필요할 때 꺼내씀

let username = "홍길동";   // 이름 저장
let count = 0;             // 숫자 저장


📌 활용: 로그인 사용자 이름, 장바구니 개수, 상태 저장

2. 조건문 & 반복문 (흐름 제어 도구)
조건문 (if)

👉 상황에 따라 다른 코드 실행

if (count > 0) {
  console.log("상품이 있습니다");
} else {
  console.log("장바구니가 비었습니다");
}

반복문 (for)

👉 같은 코드를 여러 번 실행

for (let i = 1; i <= 3; i++) {
  console.log(i + "번째 상품");
}


📌 활용: 슬라이드 자동 넘김, 리스트 출력, 폼 검증

3. 함수 (재사용 도구)

👉 자주 쓰는 동작을 묶어서 필요할 때 불러씀

function greet(name) {
  console.log("안녕 " + name);
}
greet("철수"); // "안녕 철수"


📌 활용: 버튼 클릭 시 실행, 계산 로직, 데이터 처리

4. 이벤트 (사용자 행동 감지 도구)

👉 사용자의 행동(클릭, 입력, 스크롤)에 반응

document.querySelector("button")
  .addEventListener("click", () => {
    alert("버튼 눌림!");
  });


📌 활용: 메뉴 열기/닫기, 폼 입력 감지, 스크롤 애니메이션

5. DOM 조작 (HTML 제어 도구)

👉 실제 HTML 요소를 가져와서 수정

let title = document.querySelector("h1");
title.textContent = "새 제목";      // 글자 바꾸기
title.style.color = "red";         // 색상 바꾸기
title.classList.add("active");     // 클래스 추가


📌 활용: 탭 메뉴, 모달창, 동적 리스트

6. 시각 효과 (애니메이션/스타일 제어 도구)

👉 화면 변화를 부드럽게 보여줌

let box = document.querySelector(".box");
box.style.transition = "0.5s";  
box.style.transform = "translateX(100px)";


📌 활용: 팝업 등장, 스크롤 페이드인, hover 효과 강화

🎯 어떻게 써먹는가?

변수 = 데이터 저장

조건문/반복문 = 로직 제어

함수 = 반복되는 코드를 하나로 묶음

이벤트 = 사용자의 행동을 트리거로 실행

DOM 조작 = HTML을 직접 바꿔서 화면 갱신

시각 효과 = 눈에 보이는 변화를 부드럽게 표현

➡️ 결국 이 모든 도구를 조합해서 "사용자 입력 → 화면 변화" 흐름을 만드는 것이 자바스크립트 핵심이에요.
